# Модуль на 16 Входов\выходов c опторазвязкой E3-2E 12+4IO
<img src="/MainBoard/images/Shield_4.png" width="600">

Это еще один модуль размером 1\2, на 16 входов\выходов, 12 из которых с возможностью подтяжки внешним резистором, а 4 с опто развязкой.
Как альтернатива или дополение к модулю 16IO. Его удобно использовать для подключение бинарных датчиков, с использование защиты порта через опторазвязку. 
Этот модуль половинчатый, и в корпус можно поставить одновременно 2 таких модуля (а точнее два любых с индексом 1\2).


#### Основные характеристики:
- 12 цифровых портов ввода\вывода с возможностью подтяжки внешним резистором на 10к
- 4 Цифровых порта ввода\вывода с опторязвязкой (оптопары работают только на вход)
- Для работы использует порт I2c-Shiled c основной платы. 
- Размер 1\2. Возможно использовать несколько одновременно, в том числе совмещая с другими модулями подобного формата

*Подтяжка внешним резистором нужна для стабильной работы датчиков и кнопок\включателей на больших расстояниях.*

 
<img src="/Shield%2012%2B4IO/PCB.png" width="600">
Некоторые пояснения.

- С7, С12 - Клемники используемые сверху, порты ввыода\вывода, с подтяжкой резистором 10к
- С5, С6 - Клемники с подключаемой опторазвязкой. 
- C13-C20 - Клемники для для реле. 

Адресация I2c
- Блок с пинами ввода\вывода имеет адрес 0x23
- Блок с реле - адрес 0x20

### Вариации при сборке:
Можно несколько упростить схему, если:
- Не нужна опторазвязка? Не распаиваем D3, R5-R8, N1-N4. Гребенку H паять все равно нужно. 
- Не нужна подтяжка? Убираем R1, R4, J3, J4

### Сборка платы
При пайке, у новичков могут быть сложности из-за использования smd компонентов в размерности 0805 и плотности их расположения.

Сначала паять микросхемы, потом резисторы, потом уже все остальное. Реле что бы не мешалось паяеться в самом конце.
